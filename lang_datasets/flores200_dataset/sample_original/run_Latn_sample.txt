Mu Bujapani, imisi mikuru yambere y'amashurwe y'iserasi yakorwa n'umwami w'abami ku bwiwe nyene gusa n'abandi bari mu bakuru bo ku kirimba c'i Bwami.
Imfungurwa zo muri Indoneziya zigizwe n'inteko nyinshi cane zo mu ntara zaho zitandukanye, narirya ico gihugu kigizwe n'amazinga 17.000.
Ibifungurwa vy'i Java bisigaye biboneka cane muri ayo mazinga, bikaba bigizwe n'imfungurwa zitandukanye zirimwo ibirungo bisanzwe. Ibirungo nyamukuri ab'i Java bakunda ni ivyema, amapiripiri, isukari (canecane isukari y'i Java ikorwa mu nazi) n'ibindi birungo bimota.
